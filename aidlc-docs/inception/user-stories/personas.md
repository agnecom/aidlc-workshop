# User Personas

## Persona 1: 고객 (Customer)

| 항목 | 내용 |
|------|------|
| 이름 | 김민수 (대표 고객) |
| 역할 | 매장 방문 고객 |
| 기술 수준 | 초~중급 (스마트폰/태블릿 기본 사용 가능) |
| 목표 | 대기 없이 빠르고 편리하게 음식 주문 |
| 불만 사항 | 직원 호출 대기, 복잡한 주문 절차, 주문 상태 불확실 |
| 사용 환경 | 매장 테이블에 설치된 태블릿 (브라우저 기반) |
| 핵심 니즈 | 직관적 메뉴 탐색, 간편한 주문, 실시간 상태 확인 |

**사용 시나리오**: 매장에 앉아 태블릿으로 메뉴를 탐색하고, 장바구니에 담은 후 주문을 확정합니다. 주문 후에는 현재 주문 상태를 실시간으로 확인합니다.

## Persona 2: 매장 관리자 (Admin)

| 항목 | 내용 |
|------|------|
| 이름 | 박서연 (매장 운영자) |
| 역할 | 매장 소유자 겸 운영 관리자 |
| 기술 수준 | 중급 (웹 기반 관리 도구 사용 경험) |
| 목표 | 효율적인 매장 운영, 실시간 주문 파악, 메뉴 관리 |
| 불만 사항 | 주문 누락, 수기 관리의 비효율, 테이블 상태 파악 어려움 |
| 사용 환경 | 카운터 PC 또는 태블릿 (브라우저 기반) |
| 핵심 니즈 | 실시간 주문 모니터링, 간편한 테이블/메뉴 관리 |

**사용 시나리오**: 카운터에서 실시간 대시보드로 들어오는 주문을 확인하고, 주문 상태를 변경합니다. 테이블 이용 완료 시 세션을 종료하고, 메뉴를 등록/수정합니다.
