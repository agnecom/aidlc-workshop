# Unit 4 - Frontend Components

## Admin Frontend

### TableManageView
- 테이블 목록 (번호, 활성 세션 주문 수, 총액)
- 테이블 추가 폼 (번호, 비밀번호)
- 각 테이블: 상세 보기, 이용 완료, 비밀번호 변경 버튼
- 테이블 상세: 현재 세션 주문 목록
- 과거 내역 버튼 → 날짜 필터 + 이력 목록
- data-testid: table-list, table-add-btn, table-form, table-card-{id}, table-complete-{id}, table-history-{id}
